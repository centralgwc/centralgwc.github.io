Attributes

A link

Tags can also have attributes. These include src for images and href for links. Today we'll look at two more attributes, style and class.

Inline style

You can add style information to any element by adding the style attribute and putting some CSS in it. CSS has the form "property: value;" for example: color:red;.

Style tag

It's better practice to put all of your CSS inside a style tag. That way, if you want to change how the page looks, you don't have to dig through everything. The style tag goes inside head.

Class and selectors

Here are some selectors in css:

  • * styles everything
  • A tag selector like p styles all p elements
  • A .class selector, combined with a style tag, styles a specific item or set of items
  • Inside your tag, add class="example"
  • Then use the selector .example in your style
